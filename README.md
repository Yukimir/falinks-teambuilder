## Development

Requirements:

- Node.js: 16.x (used `.at()`)
- MongoDB Atlas (used Atlas Search). Set environment variables `MONGODB_URI` and `MONGODB_APIKEY`

### Install

```sh
npm install
```

### Develop

```sh
next dev
```

### Run tests

```sh
npm run test
```
